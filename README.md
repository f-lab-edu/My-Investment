# my-investment

## 프로젝트 설명

## 🌟 재테크 관리앱 
나의 자산을 관리하며, 재테크 상황을 알려준다.
사용자들간의 재테크 방법도 공유하고 자랑한다.


## 🌟 프로젝트 사용기술
** Spring Boot
** Java
** MySQL
** Mybatis
** gradle

<br>


## 🌟 주요기능
**1. 로그인/회원가입**

**2. 달력 → 급여입력, 지출, 31일 최종금액(달 합산)**

**3. 자산페이지 -> 총 자산, 예적금,주식,기타, 현금**

**4. 메인페이지 -> 사용자 이달 자산 top3, 사용자들 이달 금액 게시판**

**5. 전달- 이번달 비교페이지**
